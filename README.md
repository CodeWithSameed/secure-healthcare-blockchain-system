# 🏥 Secure Health Care Data Storage Using Blockchain

A secure healthcare management system that leverages blockchain technology to provide integrity, transparency, and protection of sensitive medical records. The system enables authorized users to store, manage, and access healthcare data while ensuring privacy and preventing unauthorized modifications.

## 📌 Overview

Healthcare records contain sensitive information that requires strong security and reliability. This project integrates blockchain concepts with a database-driven healthcare application to ensure secure storage, tamper resistance, and controlled access to patient information.

## ✨ Features

- 🔐 Secure storage of healthcare records
- ⛓️ Blockchain-based data integrity
- 👨‍⚕️ Doctor and patient management
- 📋 Electronic medical records management
- 🔑 User authentication and authorization
- 📊 Dashboard for managing healthcare data
- 🛡️ Protection against unauthorized data modification
- 💾 Database integration for efficient record storage
